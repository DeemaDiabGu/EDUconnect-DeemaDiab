Introduction to CSE211 Web programming course

The Web programming course focuses on HTML, CSS, and JavaScript fundamentals

It aims to equip students with the skills to create and manage web applications effectively.

Course objectives

By the end of this course, students will be able to:

  1. Understand the structure and components of web pages using HTML.
  2. Style web pages effectively with CSS.
  3. Add interactivity to web pages using JavaScript.
  4. Develop responsive and user-friendly web applications.

Topics coverd

  • HTML Basics and Structure
  • CSS Styling and Layouts
  • JavaScript Fundamentals
  • Responsive Web Design
  • Web Development Best Practices
