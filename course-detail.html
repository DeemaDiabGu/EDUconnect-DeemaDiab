CSE211: Web Programming

Build your first web pages from scratch

Course Overview

This course introduces the basics of web development using HTML and CSS.

What You Will Learn

  1. How to write basic HTML tags
  2. How to style pages with simple CSS
  3. How to build multi-page websites
  4. How to use images, links and lists

Requirements

  • Basic computer skills
  • Internet connection
  • Simple text editor

Course Level: Beginner

Duration: 4 weeks 3 hours per week

Course Syllabus

Chapter 1: Getting Started with HTML

Lesson 1: Introduction to the Web

What a website is and how it works.

Lesson 2: Basic HTML Structure

Using html, head and body elements.

Chapter 2: Working with Text and Links

Lesson 1: Headings and Paragraphs

Organizing content with headings and text.

Lesson 2: Lists and Links

Creating lists and links.

Chapter 3: Introduction to CSS

Lesson 1: CSS Basics

Example: Simple styled heading

Adding color and basic styles to text and elements.

Lesson 2: Page Layout

Arranging content on the page.
